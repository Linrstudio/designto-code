## Naming

```
default: [text]
with_other_property: [text][data]
conflict_name: [key][name]
conflict_name_fallback: [name][{num}]
```

e.g.

**default**

- `text`

**with other property**

- `textData`
- `text_data`

**conflict name**

- `labelText`
- `labelTextData`

**conflict name fallback**

> This is not likely to happen.

- `labelText1`
- `label_text_1`
- `labelTextData1`
