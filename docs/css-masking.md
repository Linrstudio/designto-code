- mask-composite
- mask-image

- mask-origin
- mask-position
- mask-repeat
- mask-size

- [ ] Opera
- [ ] Safari

## Design masking to css alternative.

Interpreting masking group as a Stack, we use `hidden: true` as alternative to using css `mask-clip` + @
